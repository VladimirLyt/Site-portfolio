# Media structure

- home/main/hero-cover.svg - image for hero section on index
- home/main/featured-project.svg - big first-screen project image on index
- home/projects/shop-card.svg - card image for shop project on index
- home/projects/studio-card.svg - card image for studio project on index
- home/projects/dashboard-card.svg - card image for dashboard project on index
- home/projects/hover-studio.svg - hover image for first project card
- home/projects/hover-profession.svg - hover image for second project card
- home/projects/hover-getpatent.svg - hover image for third project card
- home/projects/hover-next.svg - hover image for fourth project card
- home/decor/decor-top.svg - decorative image for index
- home/decor/decor-bottom.svg - decorative image for index
- home/decor/circle-left.svg - first avatar circle on index
- home/decor/circle-right.svg - second avatar circle on index
- home/services/icons/row-1.svg - icon in left circle for first row
- home/services/icons/row-2.svg - icon in left circle for second row
- home/services/icons/row-3.svg - icon in left circle for third row
- home/services/logos/*.svg - logos inside service cards (replace later)
- projects/shop/cover.svg - cover image for shop project page
- projects/studio/cover.svg - cover image for studio project page
- projects/dashboard/cover.svg - cover image for dashboard project page

You can replace SVG files with JPG/PNG/WebP and keep the same names by updating paths in HTML.
